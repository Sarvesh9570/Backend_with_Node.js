This is a Node.js-based backend for managing hotel data like users (persons) and menus.

Use the following endpoints to interact with the API:

  • GET /menu - View all menu items
  • GET /menu/sweet - View all sweet items
  • GET /menu/sour - View all sour items
  • GET /menu/spicy - View all spicy items
  • POST /menu - Add a new menu item
  • GET /persons - View all registered persons -- Protected route
  • POST /persons/signup - Register a new user and receive a JWT token
  • POST /persons/login - Login and receive a JWT token

üîê For protected routes, make sure to include a valid Authorization: Bearer <token> header.
